Grid System

Pulse uses a flexible 12-column grid system built with Flexbox, ensuring your layouts are responsive across all devices.

Basic Grid

Wrap .pulse-col-* classes inside a .pulse-row. Use .pulse-container for centered content with max-width.

Col-4

Col-4
Col-6
Col-6
Col-12

    <div class="pulse-col-4">...</div>

    <div class="pulse-col-4">...</div>

Responsive Grid

Use breakpoint prefixes like pulse-col-sm-*, pulse-col-md-*, and pulse-col-lg-* to define different column widths at various screen sizes.

Col-12 (xs), Col-6 (sm), Col-4 (md+)
Col-12 (xs), Col-6 (sm), Col-4 (md+)
Col-12 (xs/sm), Col-4 (md+)
<div class="pulse-row">
    <div class="pulse-col-12 pulse-col-sm-6 pulse-col-md-4">...</div>
    <div class="pulse-col-12 pulse-col-sm-6 pulse-col-md-4">...</div>
    <div class="pulse-col-12 pulse-col-sm-12 pulse-col-md-4">...</div>
</div>

Components

Pulse offers a set of pre-built, styled components to kickstart your UI development.

Buttons

Use .pulse-btn for base styling, then add color variants like .pulse-btn-primary, .pulse-btn-success, etc. for different themes. Outline variants are also available.

<button class="pulse-btn pulse-btn-primary">Primary Button</button>
<button class="pulse-btn pulse-btn-outline-primary">Outline Primary</button>

Cards

Cards are flexible content containers. Use .pulse-card, with optional .pulse-card-header, .pulse-card-body, and .pulse-card-footer.

Card Header
This is the main content of the card. You can put any text, images, or other components here.
Card Footer
<div class="pulse-card">
    <div class="pulse-card-header">Card Header</div>
    <div class="pulse-card-body">
        This is the main content.
    </div>
    <div class="pulse-card-footer">Card Footer</div>
</div>

Forms

Standardized form elements for consistent input styling. Use .pulse-form-group to group labels and controls, and .pulse-form-control for inputs.

<div class="pulse-form-group">
    <label for="email" class="pulse-form-label">Email address</label>
    <input type="email" id="email" class="pulse-form-control" placeholder="you@example.com">
</div>
<label class="pulse-checkbox">
    <input type="checkbox"> Remember me
</label>

Alerts

Provide contextual feedback messages with predefined alert styles. Use .pulse-alert with a color variant (e.g., .pulse-alert-info).

ℹ This is an info alert! It provides general information.
✓ Success! Your operation was completed successfully.
⚠ Warning! Something needs your attention.
✖ Error! An error occurred during the process. 
<div class="pulse-alert pulse-alert-info">
    <span class="pulse-alert-icon">&#x2139;</span> This is an info alert!
</div>
